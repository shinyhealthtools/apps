ShinyHealthTools

About This Resource

ShinyHealthTools was built on a single conviction: that advanced statistical analysis should never be locked behind commercial software or require programming expertise. These 27 tools are offered freely to academics, clinicians, government researchers, and students worldwide — with no fees, no subscriptions, and no installation required.

All applications run directly in any modern web browser. Source code is openly available on GitHub under the MIT License. This site is designed to remain a useful, stable resource for health researchers for as long as possible.

"Data analysis should be simple and accessible to all." — Mudasir Mohammed Ibrahim

Browse by Research Purpose

Sample Size Calculation & Power Analysis

  • CalcuStats (Shiny) / CalcuStats (HTML) — Performs sample size calculations for known and unknown populations. Adjusts for non-response rate. Generates a publication-ready participant flowchart. Suitable for cross-sectional and other research studies.
  • R DataLab — A full-feature statistical analysis platform for descriptive and inferential statistical analyses.

Survival Analysis

  • KMPlot Genie — Kaplan-Meier survival curves with log-rank test p-values. Handles right-censored time-to-event data. Produces publication-ready plots for clinical and epidemiological manuscripts.

Mediation & Moderation Analysis

  • MedModr (Shiny) / MedModr (HTML) — Supports simple mediation, serial mediation, moderation, and moderated mediation (conditional indirect effects). Generates interactive path diagrams and simple slopes plots with embedded coefficients and p-values. Users can choose between an intuitive interactive interface or a syntax mode.

Regression & Statistical Modeling

  • Robust Regressor — Robust regression via the robustbase R package. Outlier-resistant parameter estimation with full model diagnostics.
  • RegEffect Xplorer — Visualises regression coefficients and model effects interactively. Useful for communicating regression results in manuscripts and presentations.
  • CMH Analyzer — Cochran-Mantel-Haenszel test for stratified 2×2 tables. Controls for confounding in categorical outcomes. Widely used in epidemiology and pharmaceutical research.
  • CATrend Analyzer (Shiny) / CATrend Analyzer (HTML) — Cochran-Armitage trend test for ordinal categorical data. Frequently used in dose-response and regulatory contexts.

Factor Analysis & Psychometrics

  • FactorGuard — Parallel analysis with scree plot visualization. Determines the correct number of factors to retain in EFA. More statistically rigorous than the eigenvalue-greater-than-one rule.
  • FAnalyzr — Full exploratory and confirmatory factor analysis (EFA and CFA). Supports dimensionality reduction for scale development and validation research.

Visualization & Diagnostic Tools

  • ROC Curve Builder (Shiny) / ROC Curve Builder (HTML) — ROC curves for evaluating binary diagnostic tests. Reports AUC, 95% confidence intervals, sensitivity, specificity, and optimal Youden's J threshold.
  • ggPubPlot — Publication-ready plots using ggplot2. Supports box plots, bar charts, violin plots, and scatter plots with journal-compliant formatting.
  • RegEffect Xplorer — Visual exploration of regression model effects and variable relationships.

Epidemiology & Public Health

  • EpiDem Suite — A comprehensive suite of epidemiological statistics covering relative risk, odds ratios, incidence rates, and other common public health measures. Designed for routine use in health research settings.
  • PharmaCalc Pro — Pharmaceutical calculations including dosage, formulation, and pharmacokinetic computations for clinical researchers.

Data Preparation, Conversion & Sampling

  • Data2SPSS — Converts files from R (.RData), Stata (.dta), SAS (.sas7bdat), Excel (.xlsx), and CSV formats into SPSS-compatible .sav files, preserving variable and value labels.
  • Data TransformR — Variable recoding, log transformation, standardisation, and preprocessing before model fitting.
  • CleanMyData — Data quality screening: identifies missing values, detects outliers, flags inconsistencies, and provides visual summaries of data problems.
  • QuickStatsGen — Simulates synthetic research datasets for teaching, protocol development, and pilot testing of analysis pipelines.
  • SysSampler (Shiny) / SysSampler (HTML) — Computes the systematic sampling interval (K) and generates a random starting point for structured survey sampling frameworks.
  • TagSelect — Randomisation and participant selection for research studies and clinical trials with traceable selection records.
  • APA Table Generator — Produces APA 7th-edition formatted tables from pasted or uploaded data, ready to copy directly into a manuscript.
  • Transpose Table (HTML) — Converts cross-tabulated summary data back into individual-level observations for import into SPSS, R, or Excel.
  • R DataLab — An interactive analysis workspace combining data exploration, visualisation, and modelling in a single interface.

Why Use ShinyHealthTools?

  • Validated for research. All tools are benchmarked against established software and methods. Results are suitable for grant applications, regulatory submissions, and peer-reviewed publications.
  • Fully open source. All source code is publicly available on GitHub under the MIT License. Built with R 4.3+, tidyverse, survival, psych, and other standard CRAN packages.
  • No installation required. Shiny applications run via shinyapps.io. HTML tools run directly in your browser. Works on any operating system with no administrator rights or R setup required.
  • Permanently free. No license fees, no subscriptions, no institutional access walls. All 27 tools are and will remain free for academic, government, clinical, and non-profit use.
  • No data leaves your machine. HTML-native tools execute entirely client-side. Shiny tools process data in ephemeral sessions. No personal or clinical data is retained on any server.
  • Designed for low-resource settings. Built with health researchers in sub-Saharan Africa and other resource-limited environments in mind. Works on modest hardware and slow internet connections.

About the Developer

ShinyHealthTools was created and is maintained by Mudasir Mohammed Ibrahim, a nurse researcher, data analyst, and open-source software advocate with a focus on public health research tools.

The platform grew from a recognition that many powerful statistical methods remain inaccessible to researchers who lack programming skills or institutional access to commercial software such as SPSS, SAS, SmartPLS, Mplus, or Stata. ShinyHealthTools addresses this gap by providing validated, free, browser-based alternatives to the most commonly needed health research analyses.

The project is particularly oriented toward health research settings in sub-Saharan Africa and other low- and middle-income contexts, where commercial software licensing costs can be prohibitive.

Mudasir maintains a portfolio of applications covering the full research workflow — from study design and sampling (CalcuStats, SysSampler) through data preparation (CleanMyData, DataTransformR, Data2SPSS), analysis (MedModr, KMPlot Genie, FAnalyzr, EpiDem Suite), and dissemination (ggPubPlot, APA Table Generator).
