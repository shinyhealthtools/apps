About This Website

ShinyHealthTools was built on a single conviction: that advanced statistical analysis should never be locked behind commercial software or require programming expertise. The name reflects its origins, as many of the tools were initially developed as Shiny applications to make complex analyses interactive, intuitive, and user friendly.

As the platform has evolved, practical limitations, particularly those associated with free hosting on shiny.io, have driven a transition toward more scalable solutions. Many tools have been re-engineered into lightweight, browser based HTML applications, with continuous efforts underway to translate the majority of the apps into this format. This ensures improved performance, broader accessibility, and long term sustainability. The name ShinyHealthTools is retained to represent not only its origins, but also its ongoing commitment to clarity, interactivity, and simplicity in data analysis.

These tools are offered freely to academics, clinicians, government researchers, and students worldwide, with no fees, no subscriptions, and no installation required. All applications run directly in any modern web browser. Source code is openly available on GitHub under the MIT License, supporting transparency, collaboration, and reproducibility.

This site is designed to remain a stable and valuable resource for health researchers globally, continuously evolving to meet the needs of a growing research community.

"Data analysis should be simple and accessible to all." — Mudasir Mohammed Ibrahim (Founder)

Sample Size Calculation & Power Analysis

  • CalcuStats (Shiny) / CalcuStats (HTML) — Performs sample size calculations for known and unknown populations. Adjusts for non-response rate. Generates a publication-ready participant flowchart with one click. Also includes descriptive and inferential statistics. Suitable for cross-sectional and other research studies.  [Source — Shiny]  [Source — HTML]

Survival Analysis

  • KMPlot Genie (Shiny) — Creates Kaplan-Meier survival curves with log-rank test p-values. Handles right-censored time-to-event data. Produces publication-ready plots for clinical and epidemiological manuscripts.  [Source]

Mediation & Moderation Analysis

  • MedModr (Shiny) / MedModr (HTML) — Supports simple mediation, serial mediation, moderation, and moderated mediation (conditional indirect effects). Generates interactive path diagrams and simple slopes plots with embedded coefficients and p-values. For the shiny version, users can choose between an intuitive interactive interface or a syntax mode.  [Source — Shiny]  [Source — HTML]

Regression & Statistical Modeling

  • Robust Regressor (Shiny) — Tool for robust regression via the robustbase R package. Outlier-resistant parameter estimation with full model diagnostics. Useful when standard OLS is compromised by influential observations.  [Source]
  • CMH Analyzer (Shiny) — Tool for Cochran-Mantel-Haenszel test for stratified 2×2 tables. Controls for confounding in categorical outcomes.  [Source]
  • CATrend Analyzer (Shiny) / CATrend Analyzer (HTML) — Tool for Cochran-Armitage Trend test (CATT) for ordinal categorical data with interactive chart outputs.  [Source — Shiny]  [Source — HTML]

Factor Analysis & Psychometrics

  • FactorGuard (Shiny) — Tool for parallel analysis with scree plot visualization. Determines the correct number of factors to retain in EFA. More statistically rigorous than the eigenvalue-greater-than-one rule, preventing over-extraction.  [Source]
  • FAnalyzr (Shiny) — Full exploratory and confirmatory factor analysis (EFA and CFA) tool. Supports dimensionality reduction for scale development and validation research. Includes factor loading tables and path diagrams.  [Source]

Visualization & Diagnostic Tools

  • ROC Curve Builder (Shiny) / ROC Curve Builder (HTML) — Generates ROC curves for evaluating binary diagnostic tests. Reports AUC, 95% confidence intervals, sensitivity, specificity, and other metrics.  [Source — Shiny]  [Source — HTML]
  • ggPubPlot (Shiny) — Tool for creating publication-ready plots using ggplot2. Supports box plots, bar charts, violin plots, and scatter plots with journal-compliant formatting and high-resolution export.  [Source]
  • RegEffect Xplorer (Shiny) — Visual exploration of regression model effects and variable relationships.  [Source]

Epidemiology & Public Health

  • EpiDem Suite (Shiny) — A comprehensive suite of epidemiological statistics covering relative risk, odds ratios, incidence rates, attributable risk, and other common public health measures. Designed for routine use in health research settings.  [Source]
  • PharmaCalc Pro (Shiny) — Pharmaceutical calculations including dosage, formulation, and pharmacokinetic computations for clinical researchers and healthcare professionals.  [Source]

Data Preparation, Conversion & Sampling

  • Data2SPSS (Shiny) — Converts files from R (.RData), Stata (.dta), SAS (.sas7bdat), Excel (.xlsx), and CSV formats into SPSS-compatible .sav files, preserving variable and value labels. Also converts SPSS .sav back to other formats.  [Source]
  • Data TransformR (Shiny) — Variable recoding, log transformation, standardisation, and preprocessing before model fitting. Preview transformed distributions before export.  [Source]
  • CleanMyData (Shiny) — Data quality screening: identifies missing values, detects outliers using IQR and z-score methods, flags inconsistencies, and provides visual summaries of data problems before analysis.  [Source]
  • QuickStatsGen (Shiny) — Simulates synthetic research datasets for teaching, protocol development, and pilot testing of analysis pipelines. Control over variable types, distributions, means, SD, and correlations.  [Source]
  • SysSampler (Shiny) / SysSampler (HTML) — Computes the systematic sampling interval (K) and generates a random starting point for structured survey sampling frameworks. Produces the full list of selected participant numbers for field use.  [Source — Shiny]  [Source — HTML]
  • TagSelect (Shiny) — Participant randomisation and selection tool for research studies and clinical trials with traceable, auditable selection records.  [Source]
  • APA Table Generator (Shiny) — Produces APA 7th-edition formatted tables from pasted or uploaded data, ready to copy directly into a manuscript with no reformatting required.  [Source]
  • Transpose Table (HTML) — Converts cross-tabulated (contingency table) summary data back into individual-level column-format observations for import into statistical softwares such as SPSS, R, or Excel. Ideal for re-analysing published grouped data.  [Source]
  • R DataLab (Shiny) — An interactive analysis workspace combining data exploration, visualisation, and modelling in a single interface.  [Source]

Why Use ShinyHealthTools?

  • Validated for research. All tools are benchmarked against established software and methods. Results are suitable for grant applications, regulatory submissions, and peer-reviewed publications.
  • Fully open source. All source code is publicly available on GitHub under the MIT License. Built with R 4.3+, tidyverse, survival, psych, and other standard CRAN packages.
  • No installation required. Shiny applications run via shinyapps.io. HTML tools run directly in your browser. Works on any operating system with no administrator rights or R setup required.
  • Permanently free. No license fees, no subscriptions, no institutional access walls. All 27 tools are and will remain free for academic, government, clinical, and non-profit use.
  • No data leaves your machine. HTML-native tools execute entirely client-side. Shiny tools process data in ephemeral sessions. No personal or clinical data is retained on any server.
  • Designed for low-resource settings. Built with health researchers in sub-Saharan Africa and other resource-limited environments in mind. Works on modest hardware and slow internet connections.

About the Developer

ShinyHealthTools was created and is maintained by Mudasir Mohammed Ibrahim, a Ghanaian nurse researcher, data analyst, and open-source software advocate with a focus on public health research tools.

The project is particularly oriented toward health research settings in sub-Saharan Africa and other low- and middle-income contexts, where commercial software licensing costs can be prohibitive.

Mudasir maintains a portfolio of applications covering the full research workflow — from study design and sampling (e.g., CalcuStats, SysSampler) through data preparation (CleanMyData, DataTransformR, Data2SPSS), analysis (e.g., CATrend Analyzer, MedModr, KMPlot Genie, FAnalyzr, EpiDem Suite), and dissemination (e.g., ggPubPlot, APA Table Generator).
